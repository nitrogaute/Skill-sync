# GTB Ventures — Opportunity Evaluations

## Evaluation Template v2

Combines original GTB filters with Priestley Pain/Prize/Payment and Abdaal Holy Trinity.

### Step 1: Holy Trinity Check (Pass/Fail)
- [ ] **Person:** Can you name a specific person/company who would buy this?
- [ ] **Problem:** What problem do they have that keeps them up at night?
- [ ] **Product:** What would the solution look like?

If you can't name a real person → stop. Too abstract.

### Step 4: Gut Check
- **2-year test:** Could Gaute work in this niche for 2 years?
- **No-fail scenario:** If success was guaranteed, would he choose this?
- **Value equation:** (Dream Outcome × Likelihood) ÷ (Time + Effort + Sacrifice)
- **Price on value:** Charge ~1/10 of value created. What's the 10x?

### Scoring
- **24-30:** Strong pursue — validate this weekend
- **18-23:** Promising — needs refinement, run 30 validation conversations
- **12-17:** Weak — likely pass unless a unique angle emerges
- **<12:** Hard pass
