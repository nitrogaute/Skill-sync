# Operations & Tactics

## Agent Operations

Source: Brian Castle — Structuring Jobs for Agents

- **Create jobs, not tasks.** Agents should have recurring responsibilities, not one-off prompts.
- **Three systems:**
  1. **Scheduling:** Mission control dispatching recurring work
  2. **Process:** Skills as markdown operating manuals
  3. **Output:** Artifacts stored centrally + notifications on completion
- **Start small:** One or two tasks at pennies per execution, scale when proven.
- **Lower the hiring threshold:** Agents don't need enough work for a salary — even micro-roles create value.

## Thinking System

Source: Greg Eisenberg + Internet Vin — Obsidian + Claude Code

- **Single source of truth:** Maintain a living knowledge base the agent reads from.
- **Pattern discovery:** Use cross-referencing to surface implicit ideas and connections.
- **Drift detection:** Periodically compare stated intentions vs. actual behavior/spending.
- **Manage the vault, not the agent** — keep context current, let the agent pull what it needs.

## AI Agent Services (Freelance/Agency)

Source: Nick (OpenClaw monetization video)

- **Upwork RPA arbitrage:** Find automation jobs ($500-$20k+), use AI agents to build demos, apply with completed proposals.
- **Verticalize:** Pick one industry, build reusable workflow library. Domain advantage > generalist.
- **Design thinking mapping:** Plot opportunities by value vs effort/cost/time. Target high-value, low-effort first.
- **The gap:** Businesses want automation but lack technical knowledge. Even basic agent setup is valuable.
- **Computer use agents = the unlock.** Operate legacy platforms without APIs, navigate UIs, bridge systems.
- **Avoid:** Heavily regulated industries (healthcare, finance) initially.
- **Scale:** Multiple agent instances in VMs, parallel work across tasks or clients.

## Sales Funnels

Source: Russell Brunson — DotCom Secrets (via Ali Abdaal)

- **The McDonald's model:** Zero profit on the burger (front-end). All profit from "would you like fries with that?" (back-end upsells).
- **Funnels guide customers:** awareness → first purchase → upsell → recurring. Each step multiplies revenue from existing traffic.
- Essential for lifestyle businesses built on fun, freedom, flexibility, fulfillment, and financial freedom.

Source: MJ DeMarco — The Millionaire Fast Lane (via Ali Abdaal)

## 1-Person AI Business — The Founder's Triangle

Source: "1-Person AI Business"

- **Domain + Depth + Distribution** — all three required for defensible solo business
  - Domain: 5+ years in specific industry (pain points, buying process, customer knowledge)
  - Depth: What feels like play to you but work to others
  - Distribution: Unfair access to customers (networks, partnerships, captive audiences)
- **DREAM Machine** — 5 functions every solo founder must cover:
  - **D**emand (lead gen), **R**evenue (pricing/margins), **E**ngine (core product), **A**dmin (back-office), **M**arketing (brand)
  - AI automates all 5 — 58% of small businesses already use generative AI for this
- **3 Defensive Moats:**
  1. Counterpositioning — attack competitor revenue models (Netflix vs Blockbuster late fees)
  2. Sticky habits — high switching costs (ChatGPT vs Gemini)
  3. Proprietary data loops — self-reinforcing improvement (Cursor analyzing dev keystrokes)
- **1,000x cost collapse** — $3,000 tasks now cost $3. Intelligence is a commodity.

## Building a Business Part-Time

Source: "Build a Business Part-Time" — Ali Abdaal

- **168 hours/week audit:** Map actual time. After sleep (56h), job+commute (50h), maintenance (21h) → ~41h flexible. Only need 10-15 focused hours.
- **Want > discipline.** If you don't enjoy the work, you'll burn out in 3 months. Start with 2-5h/week to test genuine interest.
- **Entertainment audit:** Cut mindless consumption. Only watch shows with friends. Frees 2h/day.
- **Weekend mornings = gold.** 2-3h Saturday + Sunday mornings = significant progress while preserving family time.
- **Time stacking:** Commute → audiobooks. Chores → business YouTube. Walking → voice notes for planning.
- **Don't over-optimize.** Family time stays sacred. Complex work needs dedicated focus blocks.

## Value Chain Decoupling (Harvard MBA)

Source: "First Lesson Harvard MBA" — Prof. Thales Teixeira

### Core Concept
- Digital disruption can be engineered by deconstructing the customer value chain
- Map every activity customers do to acquire, use, and dispose of goods/services
- Find the weakest link → break it off → steal that activity from incumbents

### Three Types of Decoupling
1. **Value creating** — separate enjoyable activities (Twitch: watch gaming without playing). Investors value these highest.
2. **Value eroding** — eliminate frustrating steps (Steam: no physical store visits for games)
3. **Value capturing** — separate payment from usage (Fortnite: free to play, pay for items)

### 5-Step Framework
1. Map complete customer value chain for any industry
2. Classify each activity: value creating, capturing, or eroding
3. Identify the weakest link (most customer dissatisfaction)
4. Break apart the chain — steal that activity from competitors
5. Anticipate incumbent responses and preempt them

### Finding Weak Links
- Dissatisfaction stems from: excessive **cost**, **time**, or **effort**
- Example: Insurance → comparing policies across companies = weakest link → InsurTech startups won
- AI opportunity: Apply AI specifically to reduce cost/time/effort at identified weak links (don't spray AI everywhere)

### Coupling (Expansion)
- After successful decoupling, add adjacent activities (Uber: rides → food delivery)
- Practice on familiar industries before attempting unfamiliar ones
