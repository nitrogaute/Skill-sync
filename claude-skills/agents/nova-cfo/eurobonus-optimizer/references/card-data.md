# Card Data Reference

Last updated: 2026-04-09

## Card 1: DNB Saga Mastercard with SAS Eurobonus Upgrade

| Field | Value |
|---|---|
| Annual fee | 2,028 kr (169 kr/month) |
| Bonus points (domestic) | 15 per 100 NOK |
| Bonus points (foreign currency) | 20 per 100 NOK |
| Level points | Threshold-based per calendar year |
| Level point thresholds | 10k at 100k, +10k at 200k, +10k at 500k spend |
| Max level points/year | 30,000 |
| Welcome bonus | 10,000 points |
| Lounge access | 4 visits/year (Norway and abroad) |
| Companion ticket | No |
| Insurance | Rental car excess, purchase/price protection |
| FX markup | ~1.75% (estimated, verify with DNB) |
| Spending cap | No cap on bonus point earning |

**Level point math:**
- Spend 100k = 10,000 level points (10 pts per 1,000 NOK effective)
- Spend 200k = 20,000 level points (10 pts per 1,000 NOK effective)
- Spend 500k = 30,000 level points (6 pts per 1,000 NOK effective)
- Between thresholds, no additional level points earned

**Best for:** Mastercard fallback when Amex not accepted, foreign currency (lower FX fee), hitting level point thresholds with planned spending.

## Card 2: SAS EuroBonus Mastercard Premium

| Field | Value |
|---|---|
| Annual fee | 2,335 kr |
| Bonus points (domestic) | 15 per 100 NOK |
| Bonus points (foreign currency) | 20 per 100 NOK |
| Bonus points (SAS.no) | 25 per 100 NOK |
| Level points | 25% of monthly bonus points earned |
| Level points minimum | 500/month when card is used |
| Level points cap | None |
| Welcome bonus | 10,000 points |
| Lounge access | No |
| Companion ticket | No |
| Insurance | Basic |
| FX markup | ~1.75% (estimated) |
| Spending cap | None (Premium has no cap; standard is capped at 200k) |

**Level point math:**
- Domestic: 15 pts/100 NOK bonus -> 3.75 level pts/100 NOK (25% of 15)
- Foreign: 20 pts/100 NOK bonus -> 5 level pts/100 NOK (25% of 20)
- SAS.no: 25 pts/100 NOK bonus -> 6.25 level pts/100 NOK (25% of 25)
- Minimum 500 level points in any month the card is used (even if spending is low)
- 12 months of minimum usage = 6,000 level points guaranteed

**Best for:** SAS.no bookings (25 pts/100 NOK is highest rate anywhere). Also useful as guaranteed minimum 500 level pts/month just by making one purchase.

## Card 3: SAS Amex Elite

| Field | Value |
|---|---|
| Annual fee | 6,900 kr |
| Bonus points | 20 per 100 NOK on ALL purchases |
| Level points | 6 per 100 NOK spent |
| Level points cap | None |
| Welcome bonus | 15,000 points |
| Companion ticket | 1 at 150k spend, 2 at 300k spend (per calendar year) |
| Travel insurance | Comprehensive (75% of trip on card required) |
| Medical coverage | Up to 2,000,000 NOK |
| Trip cancellation | Up to 50,000 NOK |
| Delayed baggage | Up to 3,000 NOK |
| Travel coverage duration | Up to 120 days |
| Supplementary cards | Free for family members (also earn 20 pts/100 NOK) |
| FX markup | 2.50% |
| Interest-free period | 45 days |
| Credit limit | Up to 400,000 NOK |

**Level point math:**
- Flat 6 per 100 NOK on everything = 60 per 1,000 NOK
- For Gold (45k level pts from card alone): need 750,000 NOK annual spend
- For Diamond (90k level pts from card alone): need 1,500,000 NOK annual spend

**Companion ticket value:**
- A companion ticket lets you book a bonus flight for two, paying points for only one
- Long-haul SAS bonus flight: ~60,000 points one-way in SAS Plus
- Companion ticket effectively saves 60,000 points = ~6,000-9,000 NOK value
- Short-haul (Europe): ~20,000 points saved = ~2,000-3,000 NOK value

**Best for:** Default card everywhere Amex is accepted (highest domestic earning rate). Companion ticket is the most valuable single perk across all three cards. Travel insurance covers the family.

## Amex Acceptance in Norway

Generally accepted: Hotels, restaurants, online shopping, SAS, Norwegian, most chains, fuel stations (Circle K, Esso), Vinmonopolet, Apple Store, IKEA.

Generally NOT accepted: Rema 1000, Kiwi, Coop, Bunnpris, some small shops, parking meters, some vending machines, Vipps payments.

Rule of thumb: always carry a Mastercard as backup. Use Amex first, fall back to MC if declined.
